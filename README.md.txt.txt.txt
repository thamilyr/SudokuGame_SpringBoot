# Sudoku Game (Java Console)

Jogo de Sudoku via terminal com menu interativo.
Possui suporte a números fixos via argumentos do main.

## Como rodar

javac SudokuGame.java
java SudokuGame 0,1,5 3,4,9 8,8,7

## Funcionalidades
- Iniciar novo jogo
- Colocar número
- Remover número
- Visualizar
- Verificar status
- Limpar
- Finalizar
